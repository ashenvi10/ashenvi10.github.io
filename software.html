Software

Python package cegpy:
cegpy is a Python Library for creating, learning, and manipulating Chain Event Graphs from pandas dataframes. It supports modelling with asymmetric processes, i.e. processes that have structural zeros or structural missing values.
The package is still under development. The code repository can be found here. Developers:
